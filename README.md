# matlab algorithm

收藏一些日常 matlab 代码，打包成函数供大家使用。



# List

- DouglasPeucker

  > 道格拉斯-普克算法(Douglas–Peucker algorithm，亦称为拉默-道格拉斯-普克算法、迭代适应点算法、分裂与合并算法)是将曲线近似表示为一系列点，并减少点的数量的一种[算法](https://baike.baidu.com/item/算法/209025)。它的优点是具有平移和旋转不变性，给定曲线与阈值后，抽样结果一定。

- bar2line

  > 用于将柱状图转换为折线图。
